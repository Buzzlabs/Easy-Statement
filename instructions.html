Instruções

Se você não possui um projeto no Conta Aberta, não é possivel extrair seus os dados com esta extensão
Primeiro é necessário está logado no Conta Aberta. Caso não esteja logado ainda faça login na sua conta.
Se você tem um projeto e está logado no Conta Aberta, Extrair seus dados bancários é um procedimento simples:
  1. Faça login na sua conta do Itaú.
  2. Acesse a sua página de extrato.
  3. Preencha o campo com o ID do projeto para onde os dados serão enviados.
  4. Clique em 'Enviar' e pronto!
Caso tenha mais de uma o página o extrato do projeto, cada página devera ser enviada individualmente.
